1. Graceful degradation and service workers go hand in hand because both are about keeping an app usable when the ideal runtime conditions break down. Graceful degradation says: start with the full, modern experience, then make sure the app still works when something is missing. Service workers are one of the main tools that make that fallback possible on the web: they sit between the page and the network, intercept fetches, and serve cached responses when the network is unavailable or slow, so the user still sees their recipes instead of a blank page or a dinosaur.
